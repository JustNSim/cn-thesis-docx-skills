# Body Numbering Style

Use this rule for non-heading numbering in Chinese academic reports. Do not use these forms as Word heading levels, and do not mix them with manually typed heading numbers.

1. **Top body point**: use `（一）` only for a major point inside a section.
   - Put `（一）` at the start of a standalone lead sentence or short lead paragraph.
   - Use the following paragraph(s) to explain the lead sentence.
   - Do not continue immediately with `（二）` in the same paragraph.

2. **Second-level body point**: use `（1）`, `（2）`, `（3）`.
   - Use this under a `（一）` point or inside one coherent paragraph.
   - It may stay inline when each item is short, e.g. `（1）...；（2）...；（3）...。`
   - If each item needs more than one sentence, split items into separate paragraphs.

3. **Third-level body point**: use `a.`, `b.`, `c.`.
   - Use only inside `（1）`-level explanations.
   - Keep each item short and local; if it becomes long, promote it to `（1）`.

Avoid these patterns:

- `一、` or `1.` in body text when the same document already uses numbered headings.
- A `（一）` paragraph that behaves like a heading but is formatted as body text.
- Mixing `（1）`, `1)`, `①`, and `a.` at the same logical level.
- Hand-numbering Word headings; use template heading styles and automatic TOC fields.
